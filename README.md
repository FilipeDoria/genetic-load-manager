# Genetic Load Manager - HACS Integration

A Home Assistant integration for intelligent load management using genetic algorithms.

## Installation

1. Add this repository to HACS as a custom integration
2. Search for "Genetic Load Manager" in HACS
3. Click Download
4. Restart Home Assistant
5. Add the integration via Settings > Devices & Services

## Features

- Genetic algorithm-based load optimization
- Configurable entity monitoring
- Real-time optimization status
- HACS-compliant integration structure
